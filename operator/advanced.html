ESPWR

Control your ESP32 GPIO pins via HTTP requests.

  • Home
  • ADVANCED CONTROL

Use the buttons below to control the power and reset pins of your server. You can also use the following HTTP requests to control the pins:

  • api/pwr?set=high or api/rst?set=high to set pin HIGH
  • api/pwr?set=low or api/rst?set=low to set pin LOW
  • api/pwr?hold=ms or api/rst?hold=ms with hold=ms to set pin HIGH for ms milliseconds
